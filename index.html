MetaMask Steps

PC

  1. Open MetaMask and select Networks.
  2. Choose Add network and select Add a network manually.
  3. Enter the exact fields from the config panel.
  4. Save and switch to Ethernova.

Mobile

  1. Open MetaMask and tap the network selector.
  2. Select Add network and choose Manual input.
  3. Paste the same fields and save.
  4. Confirm the chain ID is 77777.

Mainnet Upgrade (NO Chain Reset)

Upgrade with the scripts below. Do not reset your chain.

Windows

scripts\apply-upgrade-mainnet.bat

Linux

./scripts/apply-upgrade-mainnet.sh

Behind the scenes

ethernova --datadir <your-datadir> init genesis-upgrade-60000.json
Warning: Do NOT replace the genesis inside your datadir. Init updates the stored chain config while preserving the genesis hash.
